Selected work

  • NCP : Neuro-Cybernetic Protocol: a safety-gated, provenance-first wire contract that lets a spiking neural simulation (NEST) perceive and act through robots, UAVs and analysis clients. Rust SDK with Python, C/C++ and TypeScript peers.
  • crebain : Adaptive Response & Awareness System (ARAS): tactical visualization & autonomy: sensor fusion, ML detection, drone physics, ROS / Gazebo. Tauri + React + Rust.
  • melkor : Gaussian-splatting & depth-analysis pipelines for 3D scene reconstruction: multi-view capture to dense geometry, CUDA-accelerated. Python & C++.
  • cortexel : agent-consumable scientific-visualization library for neural simulations: emit a declarative VizSpec and render spike rasters, Brunel networks & STDP curves, with provenance enforced fail-closed. TypeScript.
  • pid-rs : Partial Information Decomposition with continuous mutual-information (KSG) estimators, in safe Rust.
  • cobot-atlas : 3D mesh-generation pipeline: 2,023 meshes for collaborative-robot simulation and embodied AI; dataset published on Hugging Face. Python.
  • engramprivate : a NEST-based neural-modeling hub: the "brain" the rest of the stack connects to.
  • prisomaprivate : a prism for embodied agents, robotics and world models: refracts Vision-Language-Action policies through several natively-implemented analysis methods, Partial Information Decomposition among them.
